# AWS Icons

These service icons were copied from the official AWS Architecture Icons package:
https://aws.amazon.com/architecture/icons/

Only the SVGs used by the workshop cards are kept in `services/` so the app can run offline without external libraries or runtime asset requests.

Notes:
- `AWS DMS` uses the official `AWS Database Migration Service` icon.
- `Amazon MSK` uses the official `Amazon Managed Streaming for Apache Kafka` icon.
- `Amazon QuickSight` uses the package's current `Amazon Quick Suite` icon.
- `Apache Iceberg on AWS` uses the nearest official S3 Tables resource icon.
- `SageMaker Feature Store` uses the current SageMaker AI service icon because a separate Feature Store service icon was not present in the package.
